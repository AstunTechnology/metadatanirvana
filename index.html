GeoNetwork Overview

GeoNetwork is the leading open source, standards-compliant metadata catalog, used globally for national and international spatial data infrastructures. It is run by a Project Steering Committee, comprising lead developers and users from around the world.

It's also an Open Source Geospatial Foundation project, which means it has passed a stringent incubation process focusing on code quality and sustainability. This in turn means it is reliable, has regular releases, and a strong support network.

GeoNetwork can be used for the creation and storage of both spatial and non-spatial metadata, which means it can act as a single point of truth for all your metadata requirements. It also provides endpoints for sharing metadata in both human and machine-readable formats.

It provides a powerful interface for discovering and viewing metadata and data resources, and the production and editing of metadata records. We're going to take a look at some of the less well-known features in GeoNetwork that help you create high-quality metadata quickly and easily.
